# **App Name**: Envíos DosRuedas | Logística E-commerce y Mensajería MDP

## Core Features:

- Intelligent Quote Engine: An AI-powered tool that reasons through shipping dimensions and urgency to provide an optimized cost estimate for express and flex deliveries.
- Real-Time Tracking UI: A streamlined interface for customers to monitor their deliveries via unique shipment identifiers.
- Logistics Inquiry Chatbot: An AI assistant tool that handles common queries about the shipping zones, transit times, and service levels.
- Service Level Selection: An intuitive booking workflow for customers to choose between Express, LowCost, Flex, and E-Commerce plans.
- WhatsApp Direct Connect: Quick-action interface to initiate support or custom delivery requests via the verified business phone number.
- Enterprise Contact System: Specialized lead capture forms for 3PL and E-commerce high-volume partner inquiries.
- Smart Routing & API Integration: A system that optimizes delivery routes in real-time and allows store owners to connect their online shops directly for automated fulfillment.
- Live Analytics Dashboard: A visual reporting tool for business partners to track performance metrics and real-time delivery status.

## Style Guidelines:

- Primary Color: Deep Corporate Blue (#1E3A8A) for reliability and trust.
- Secondary Color: Agile Sky Blue (#3B82F6) representing speed and movement.
- Accent Color: Energetic Yellow (#EAB308) for call-to-actions, alerts, and highlighting urgency.
- Headline Font: 'Anton' (Sans-serif) for high-impact, bold, and modern display text.
- Subheadline Font: 'Bebas Neue' (Sans-serif) for category titles, badges, and secondary headings.
- Neobrutalist influence featuring a grid background pattern, card-based components with 6px hard shadows, and bold borders.
- Micro-interactions including pulsing status indicators for 'In Transit' packages and phone icons that ring on hover.
